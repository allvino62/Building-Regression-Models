Separate Predictors and Target Variable¶

¶

Rings is the target variable because it is used to estimate the age of the abalone. All other variables are used as predictors.

Convert Categorical Variable into Dummy Variables¶

In [34]:
X = pd.get_dummies(X, drop_first=True)

The variable Sex is categorical, so it must be converted into numeric dummy variables before it can be used in regression models.¶

drop_first=True removes one category to avoid redundancy.

Split the Dataset into Training and Validation Sets¶

In [35]:
X_train, X_valid, y_train, y_valid = train_test_split(

This separates the data into training and validation sets.¶

The model learns from 80% of the data and is tested on the remaining 20%.

The model learns the relationship between the predictors and Rings, then makes predictions on the validation data.

Random Forest uses many decision trees and can capture nonlinear relationships better than simple linear regression.

Create a Residual Plot for Linear Regression¶

In [14]:
residuals = y_valid - lr_pred

plt.scatter(lr_pred, residuals)
plt.axhline(y=0, linestyle="--")
plt.xlabel("Predicted Rings")
plt.ylabel("Residuals")
plt.title("Residual Plot for Linear Regression")
plt.show()

A residual plot helps check the linearity assumption.¶

If residuals are randomly scattered around zero, the linear model is more appropriate. Patterns may indicate nonlinearity or unequal variance.

Create a Q-Q Plot of Residuals¶

In [15]:
import scipy.stats as stats

stats.probplot(residuals, dist="norm", plot=plt)
plt.title("Q-Q Plot of Linear Regression Residuals")
plt.show()

The Q-Q plot checks whether residuals are approximately normally distributed.¶

If the points follow the diagonal line closely, the normality assumption is reasonable.
